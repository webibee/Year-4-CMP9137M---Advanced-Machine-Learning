# Advanced Machine Learning: Image-Text Matching & Deep Reinforcement Learning
- Course: CMP9137M – Advanced Machine Learning
- Institution: University of Lincoln
- Grade Received: 65%
- Language: Python
- Frameworks: TensorFlow/Keras, PyTorch, Stable-Baselines3, Transformers

# Overview
This project comprises two distinct machine learning tasks:

## Task 1: Image-Text Matching (Binary Classification)
Predict whether an image and a textual description match. Three classifiers were implemented and compared:
- Convolutional Neural Network (CNN)
- BERT (Bidirectional Encoder Representations from Transformers)
- Transformer (encoder-only architecture)

## Task 2: Deep Reinforcement Learning – Super Mario Bros
Train autonomous agents to play Super Mario Bros (SuperMarioBros2-v1) using three DRL algorithms:
- DQN (Deep Q-Network) – Value-based, replay buffer, target network
- A2C (Advantage Actor-Critic) – Policy-based, CPU-optimised
- PPO (Proximal Policy Optimisation) – Clipped surrogate objective, trust region
